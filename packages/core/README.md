> munan-cli 脚手架核心包

## Prerequisites

- node >=v14.16.0

## Install

```sh
npm install @munan-cli/core -g
```

## Run tests

```sh
npm run test
```

## Command

```sh
Options:
  -V, --version                    查看脚手架版本  
  -d --debug                       打开调试模式    
  -h, --help                       查看帮助信息    

Commands:
  init [options] [name]            初始化项目      
  template [options] [moduleName]  创建模板        
  help [command]                   查看命令帮助信息
```
